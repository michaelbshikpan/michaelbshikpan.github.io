# Step-by-step guide for publishing this website

## A. Create your GitHub access

1. Go to GitHub and create a free personal account.
2. Verify your email address.
3. Save your username and password securely. Do not share the password with anyone.

## B. Create the website repository

1. Click the `+` button in GitHub.
2. Choose `New repository`.
3. Name the repository exactly like this:

   `YOUR-GITHUB-USERNAME.github.io`

   Example: if your GitHub username is `smbulkaam`, name it:

   `smbulkaam.github.io`

4. Make the repository `Public`.
5. Click `Create repository`.

## C. Upload the website package

1. Unzip the package from ChatGPT.
2. Open the new GitHub repository.
3. Click `Add file`.
4. Click `Upload files`.
5. Drag all unzipped files and folders into GitHub.
6. Click `Commit changes`.

Important: upload the contents of the folder, not the folder itself. GitHub should show `index.html` at the top level of the repository.

## D. Turn on GitHub Pages

1. In the repository, click `Settings`.
2. Click `Pages` in the left menu.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch: `main`.
5. Select folder: `/ (root)`.
6. Click `Save`.

Your website address will be:

`https://YOUR-GITHUB-USERNAME.github.io/`

## E. Update the CV later

1. Open the repository.
2. Go to `assets`.
3. Delete or replace `Michael_Bulkaam_Shikpan_CV.pdf`.
4. Upload the new CV using the same filename.
5. Commit the change.

## F. Update homepage text later

1. Open `index.html`.
2. Click the pencil/edit button.
3. Edit the text.
4. Click `Commit changes`.

## G. Recommended safety practice

Do not upload private documents, phone numbers, passport information, student numbers, immigration documents, bank details, or home address information to a public website.
